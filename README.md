# BloodBank Management System Application

This is a simple desktop application for managing blood bank operations, built using Java and the NetBeans IDE.

---

## üìã Features

* **Welcome Screen:** A main dashboard to navigate the application.
* **Donor Registration:** Allows new users to register as blood donors.
* **Request Blood:** Provides a feature for users to search for and request blood from the donor database.
* **Edit Records:** Includes functionality to update and manage existing donor or request records.

---

## üöÄ How to Run

This is an Apache NetBeans project. To run it, you can:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/BloodBank.git](https://github.com/your-username/BloodBank.git)
    ```
2.  **Open in NetBeans:**
    * Open Apache NetBeans IDE.
    * Go to `File` > `Open Project...`
    * Navigate to the cloned `BloodBank` folder and select it.
3.  **Run the Project:**
    * Right-click on the `BloodBank` project in the project explorer.
    * Click `Run` (or press F6).

Alternatively, you can run the pre-compiled Java archive (`.jar`) file:

1.  Navigate to the `/dist` folder.
2.  Run the `BloodBank.jar` file (requires Java to be installed on your system).
